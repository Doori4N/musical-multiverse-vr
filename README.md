# musical-multiverse-vr

### Installation
```
npm i
```

### Run
Client
```
npm run dev
```
Server
```
PORT=4444 node ./bin/server.js
```

port by pass (make sure clients are in the same adress ip)
```
npm run dev -- --host 0.0.0.0
```
